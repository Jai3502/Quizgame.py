# Quizgame.py
A simple Python console-based quiz game that asks users common abbreviation questions (CPU, GDP, RBI, etc.) and checks their answers interactively.

# Quiz Game in Python 

This is a simple **console-based quiz game** built using Python.  
The program asks the user if they want to play and then presents a series of questions related to common abbreviations. Based on the user's input, it displays whether the answer is correct or incorrect.

---

## Features

- Interactive command-line quiz
- Beginner-friendly Python code
- Uses basic concepts like:
  - `input()`
  - `if-else` conditions
  - String comparison
- No external libraries required

---

##  Questions Included

- CPU – Central Processing Unit  
- FIR – First Information Report  
- CA – Chartered Accountant  
- GDP – Gross Domestic Product  
- UK – United Kingdom  
- RBI – Reserve Bank of India  
- SEBI – Securities and Exchange Board of India  
- UN – United Nations  

---

##  How to Run the Program

1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Open a terminal or command prompt.
4. Run the following command:

   ```bash
    python quizgame.py

* type yes when asked to start the quiz.

* Answer the questions as they appear.
